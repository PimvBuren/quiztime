# quiztime
We maken een voetbalquiz met behulp van PHP, HTML en CSS. De quiz bestaat uit vragen die we in een datta base opslaan. De HTML-structuur toont de vragen en antwoorden in een formulier, terwijl CSS zorgt voor een mooie opmaak.
